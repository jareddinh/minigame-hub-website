# minigame-hub-website

Minigame Hub is a browser-based web application featuring multiple interactive mini-games built using vanilla HTML, CSS, and JavaScript. The project focuses on DOM manipulation, game logic implementation, and responsive front-end design.

---

## Tech Stack
- HTML5
- CSS3 (including responsive media queries)
- JavaScript (vanilla, no frameworks)

---

## Features
- Tic Tac Toe game with win/draw detection
- Memory card matching game
- Number guessing game with feedback system
- Rock Paper Scissors game with randomised logic
- Responsive navigation bar
- Client-side search filtering for game selection using partial text matching
- Contact form with validation
- Responsive design using CSS media queries for mobile, tablet, and desktop layouts

---

## Key Skills Demonstrated
- DOM manipulation and event handling in JavaScript
- Game logic implementation and state management
- Responsive UI design using CSS media queries
- Form validation and user input handling
- Structuring a multi-page front-end application

---

## Usage
1. Download or clone the repository
2. Open `index.html` in a web browser

---

## Responsive Design
The application is optimised for:
- Mobile devices
- Tablets (≥ 600px)
- Desktop screens (≥ 1000px)
